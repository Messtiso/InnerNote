# InnerNote - Transparent Mood Insights from Everyday Journaling

InnerNote is a web-based journaling application designed to provide transparent mood insights from everyday writing.

## Project Goal

The aim of InnerNote is to help users reflect on their emotions through journaling while providing simple, explainable insights into emotional patterns detected in their writing.

## Tech Stack

Frontend: React  
Backend: Node.js + Express  
Database: MongoDB  
NLP: JavaScript sentiment analysis library
